    @bartificer/linkify. LinkData

    A class to represent data available for use when generating an output link with a template.

    Constructor

    Fcnew LinkData(url, textopt, descriptionopt)

    Parameters:
    Name Type Attributes Description
    url string

    the URL the link will go to.
    text string <optional>

    the link text, defaults to URL.
    description string <optional>

    a description for the link, defaults to the link text.
